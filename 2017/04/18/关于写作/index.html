写作的意义

2017-04-18

生活 写作

这两天在看 得到app 上的知识干货，让我记忆最深刻的一个概念就是『认知』。有一篇文章提到人与人之间最大的差别是认知，我这才意识到那些牛人们之所以牛是因为站在了你没到达的高度上，看到了你没看到的东西，知道了你不知道的东西，他们对这个世界的了解程度远高于你。所以在做一些决策的时候往往看的更高更远，而且他们一直都在认知升级的道路上。

得到app 是罗辑思维团队出品的一款知识应用，每天我都会阅读一期订阅的知识专栏。这段时间以来，觉得自己的思维被颠覆了不少，也学习了不少的概念，同时也纠正了不少的概念，『认知』也得到了小幅的升级。文章虽然看了很多，但是要把那些信息转化成自己的知识，需要反复的理解思考和践行，做到知行合一。而写作是一个巩固知识、记录成长的好方法。

写作，我的简单理解是书写加创作，单纯的搬运知识那跟做笔记没什么区别，写作内容就应该是独一无二的东西，要有你的灵魂在里面，而且他必须是有用的东西。

背景

我每天都要数字阅读上花费比较多的时间，最近也收藏了不少小众网站，比如少数派、利器。内容大多是一些极客和艺术创作者们的一些知识和技巧分享，非常酷炫，文章的排版也很精美，也学到了文章之外的一些知识。除了内容要吸引读者之外，网站的设计也是一大亮点，良好的阅读体验能让你有更大的阅读欲望。

想法来了，我也要成为内容生产者，你的内容能得到别人的认可无疑能带给你伟大的成就感，坚持写作能让自己看得见自己的过去，更能掌控生活的节奏。

文章的发布渠道

第三方的写作平台众多，目前比较热门的是简书和微信公众号平台，大量的作者也都会把最新的文章同步发布在各个平台上。这些平台的好处就是活跃用户比较多，你的文章被发现的几率比较大。但同样内容多了，优质资源难以找到，而且容易被一些其他的文章转移注意力，不知不觉在上面花去了很多时间。

很多写作者会自己搭建一个网站，网站的风格有着个人的鲜明的特色，通过 RSS 订阅可随时接受他们发布的新文章。这样一来的几点好处就是：

  • 网站可以按照自己的心情去布置，不受第三方平台的约束
  • 你能掌握网站的建立和维护相关的知识
  • 建立个人的品牌形象

GitHub Pages

GitHub是一个通过Git进行版本控制的软件源代码托管服务，GitHub Pages是官方推荐用来部署个人网站的一个服务，而且是免费的，可以用来记录自己的学习经历和作品展示。GitHub Pages官方是推荐使用jekyll来部署个人网站的，而比较简单上手的则是hexo，有中文教程，主题丰富并且可以直接套用。

这几天尝试着利用 Hexo 搭建了自己的第一个博客网站，还是比较顺利的，主题看了很多，最终选择了一款比较喜欢的风格Even 主题，感谢主题的作者ahonn。

网站基本要素已经有了，接下来就是给他填充内容，在这里我会记录下自己get到的最新技能，生活的感悟，还有关于技术的分享，争取让写作成为一种乐趣！

让我们拭目以待吧！
